# チュートリアル

ここでは、最も基本的な描画手順を説明します。  
正規化などの計算処理は行わず、入力したシグナル強度ファイルをもとに、そのまま画像を生成します。

描画までの基本的な流れは以下のとおりです。

1. ファイルのロード  
2. 描画データの設定  
3. 描画の実行

---

## 1. ファイルのロード
Compass から出力されたシグナル強度ファイルを読み込みます。  
`1. Import Data` と表示された枠をクリックすると、エクスプローラ（macOS の場合は Finder）が開くので、読み込むファイルを選択してください。

![upload](./images_tutorial/load_file.png)

---

## 2. 描画設定
通常はこの段階でレーン設定や描画オプションを調整します。  
しかし、デフォルトの設定のままでも描画が可能なため、本チュートリアルでは設定を変更せずに進みます。  
（設定を変えずに、そのまま画面の最下部までスクロールしてください）

---

## 3. 描画の実行
画面最下部に `Generate` ボタンがあります。  
これをクリックすると、画像が生成されます。

![generate_button](./images_tutorial/generate_image.png)

---

## 4. 画像の調整
出力された画像は、`Draw Settings` 内で調整することができます。
